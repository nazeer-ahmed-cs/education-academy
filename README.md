# Education Academy

A static informational website for Education Academy — showcasing courses, success stories, and contact information for an educational institution.

## Features

- Course listings (MDCAT, NTS, SPSCS, STS)
- Success stories with student testimonials
- About section with institution details
- Contact information
- Responsive design

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Deployed on Vercel

## Live Site

[https://education-academy-blush.vercel.app](https://education-academy-blush.vercel.app)

## Project Structure

```
├── index.html            # Main landing page
├── embed_images.py       # Image embedding script
├── images/               # Instructor and student photos
└── success_images/       # Success story images
```
